ASPICE TRACEBILITY FRAMEWORK

Framework is dedicated for creating system, software requirements, tests specification and for linking everything easily to have good tracebility including architecture design and code. Practical framework usage is visible with example project: https://www.twojkomputerowiec.bydgoszcz.pl/aspice
